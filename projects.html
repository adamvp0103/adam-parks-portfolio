Candy Cavern

Candy Cavern is a fictitious online candy store. With this project, I wanted to demonstrate the implementation of common e-commerce features like adding items to and from a cart, calculating and presenting totals based on product data, and controlled forms with intuitive input guidance. Candy Cavern also demonstrates my familiarity with the popular and powerful React library, with which I use both hooks and props for state management, React Router for page navigation, and the built-in context API for globally referenced data.

BitBabble

BitBabble is a fake developer-themed blog site. This project demonstrates my ability to work with data that is not my own. Courtesy of DummyJSON, I used the third-party API to consume data and translate it into a functional and visually unique site. The site includes some major blog features like browsing posts, detail pages for posts and users via URL parameters, liking and disliking posts, following and unfollowing users, searching for posts and users, making your own posts, and editing account information. I took advantage of React's powerful context provider architecture to manage the complex interactions between post data and user data.
